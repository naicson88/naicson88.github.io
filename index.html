Sobre a Aplicação:

O intuito desta aplicação é registrar os erros mais frequentes no CaixaExpresso, WebLogin, NPEG, ou qualquer outra rotina muito utilizada, juntamente com as possíveis soluções desses erros.

Além de consultar os registros mais frequentes, é possível também cadastrar novos registros a medida que o usuário desta aplicação achar conviniente

Os Projetos

Resumo sobre os principais projetos utilizados
  • Caixa Expresso

    Utilizado nas lojas para agilizar o processo de venda, com comandos simples e específicos para facilitar o uso.

  • WebLogin

    Principal sistema da empresa, integra dados das áreas de Vendas, Finanças, Estoque, Compras e etc

  • WebSincronismo

    Aplicação responsável por monitorar o sincronismo entre o banco de dados das unidades locais com o banco da matriz

  • NPEG

    Software responsável por gerencialmento de notas Fiscais, inclusive emissão. Não é desenvolvido pela Login
